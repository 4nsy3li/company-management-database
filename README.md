# Company Management Database

Oracle-based company management database project demonstrating relational modeling, PL/SQL business logic, and reporting.

## Included scripts
- sql/schema.sql
- sql/seed_data.sql
- sql/views.sql
- sql/procedures_and_functions.sql
- sql/analytics_queries.sql
- sql/test_queries.sql
- sql/apex_app_export.sql

## How to run
1) unzip/company-management-database
2) Run schema.sql to create tables/constraints.
3) Load seed_data.sql.
4) Run procedures_and_functions.sql.
5) Run analytics_queries.sql for reporting examples.
